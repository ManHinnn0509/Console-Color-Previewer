# Console-Color-Previewer

See how a color would look like in console. <br>
**No dependencies / requirements required.** <br>
Just run the [Main.java](Main.java) and enter the RGB / Hex code and there you have it.

## Examples of input

- `0 255 0`
- `00FF00`
- `#00FF00`
